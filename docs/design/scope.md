# OpenVN Scope

## In Scope

- Ink as the authoring language
- OpenVN intermediate story format
- Compiler
- Asset packaging
- Runtime API
- Ren'Py runtime
- Native Amiga runtime
- ARexx-first architecture
- AI-assisted content pipeline

## Out of Scope

- Point-and-click adventure engines
- Inventory systems
- Walkable areas
- Path finding
- Physics
- Tile maps
- RPG mechanics
- 3D rendering

OpenVN intentionally focuses on visual novels.
