# OpenVN Philosophy

- Author once.
- Run everywhere.
- Story first.
- Open specifications.
- Long-term compatibility.
- Small runtimes.
- Compiler over engine.

## Amiga Principle

Use ARexx whenever practical.

Native C is limited to:

- Graphics
- Audio
- Input
- File I/O
- Memory management

ACE is optional and must never be a required dependency.
